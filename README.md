# UI Component Generator

Ferramenta que usa IA (Claude) para gerar componentes React + TypeScript + Tailwind a partir de uma descrição em linguagem natural.

## Demo

[🔗 Ver projeto ao vivo](https://sua-url.vercel.app)

## Como funciona

1. Descreva o componente que precisa em português ou inglês
2. A IA gera o código React + TypeScript pronto para usar
3. Copie e cole direto no seu projeto

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Anthropic Claude API

## Rodando localmente
```bash
npm install
npm run dev
```

Crie um arquivo `.env` na raiz com:
```
VITE_ANTHROPIC_API_KEY=sua_chave_aqui
```
